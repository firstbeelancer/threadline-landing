# THREADLINE — Landing

Static marketing site for [threadline.tigerapps.pro](https://threadline.tigerapps.pro),
built with [Astro](https://astro.build) for SEO and fast static delivery.

## Stack
- Astro 4 (static site generation)
- Plain HTML/CSS (no JS framework on the client)
- Brand tokens: mint `#42DFC1` → deep purple `#6D4ADE` → indigo `#420881`

## Develop
\`\`\`
npm install
npm run dev
\`\`\`

## Build
\`\`\`
npm run build       # outputs static site to dist/
\`\`\`

## Deploy
Output `dist/` is served by Coolify as a static site. Push to `main` to redeploy.
